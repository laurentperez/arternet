CSS Houdini

tl;dr : pour un accès direct au moteur CSS via JS

plus de hacks, on a un véritable accès au moteur

objet typé, propriétés, peinture, animation, layout, fontes

Travail commencé circa 2017

Division en 6 grandes APIs

https://github.com/w3c/css-houdini-drafts

  • CSSOM API (CAPI)
  • Properties/Values API (PVAPI)
  • Paint API (PAPI)
  • Web Animation API (WAAPI)
  • Layout API (LAPI): au delà de flex et grid, pour des UI entièrement custom (UI militaire, dash voiture/avion, presse, ...)
  • Font Metrics API

Paint API

Introduction de PaintWorklet : analogie avec les ServiceWorkers pour le cache/offline/push notifs

Une Worklet est short-lived, destructible par le moteur, déplaçable entre threads

But : qu'il ne tourne pas dans la main thread (main JS UI =lag)

La Paint API donne un Canvas (2DContext)
